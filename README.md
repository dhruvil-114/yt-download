# 🚀 Python 4K/8K Video Downloader (Episode 01)

A high-performance, professional-grade video downloader built using **Python 3.14** and **CustomTkinter**. This tool deep-scans metadata to unlock every available resolution, from 144p up to pure 8K, and automatically merges them into a universal MP4 format.

---

## ✨ Key Features

* **Ultra HD Support:** Deep-scans and extracts high-resolution streams (4K/8K).
* **Native MP4 Merging:** Uses FFmpeg to merge video and audio into a single MP4 file automatically.
* **Dynamic Resolution Scraper:** Fetches all available qualities for any specific video on the fly.
* **Max Speed:** No speed throttling—downloads as fast as your internet connection allows.
* **Modern UI:** Built with a sleek, dark-themed GUI for a professional developer experience.

---

## 🛠️ Tech Stack

* **Language:** Python 3.14
* **Backend:** `yt-dlp` (Industry standard for high-speed extraction)
* **Frontend:** `customtkinter` (For the modern aesthetic)
* **Processing:** `FFmpeg` (For seamless format conversion and merging)
