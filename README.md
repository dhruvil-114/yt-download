Python 4K/8K Video Downloader (Episode 01)
A high-performance, professional-grade video downloader built using Python 3.14 and CustomTkinter. This tool deep-scans metadata to unlock every available resolution, from 144p to pure 8K, and automatically merges them into a clean MP4 format.

✨ Key Features:
8K Support: Deep-scans and extracts high-resolution streams (4K/8K).

Native MP4 Merging: Automatically uses FFmpeg to merge video and audio into a single MP4 file.

Dynamic Resolution Scraper: Automatically fetches all available qualities for any specific video.

Blazing Fast: No speed throttling—downloads as fast as your internet connection allows.

Modern UI: Built with a sleek, dark-themed GUI for a professional user experience.

🛠️ Tech Stack:
Language: Python 3.14

Library: yt-dlp (For high-speed extraction)

UI: customtkinter (For the modern aesthetic)

Engine: FFmpeg (For seamless format conversion and merging)

⚙️ Installation:
Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/yt-master-v1.4.git
Install dependencies:

Bash
pip install -r requirements.txt
Install FFmpeg:
Make sure FFmpeg is installed and added to your system's PATH for the MP4 merging feature to work correctly.

🚀 Usage:
Run the script using the following command:

Bash
python main.py
Paste your YouTube link, click Fetch, select your desired quality, and hit Download!
